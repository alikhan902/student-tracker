Student Tracker — Документация

Добро пожаловать в официальную документацию проекта Student Tracker. Этот репозиторий содержит три основных компонента:

- `frontend/` — веб-интерфейс на Vite + React
- `backend/` — сервис на Spring Boot
- `desktop/` — десктопное приложение на Java (JavaFX)

Структура документации:

- [frontend](frontend/README.md)
- [backend](backend/README.md)
- [desktop](desktop/README.md)

Как использовать эту документацию:

1. Открой нужный каталог (frontend/backend/desktop).
2. Следуй инструкциям по установке, запуску, тестированию и деплою.

Если чего-то не хватает — открой issue или сделай PR с дополнением.
