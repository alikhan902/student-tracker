# Backend — REST API (обзор)

Этот файл даёт обзор основных REST-эндпоинтов. Для точной спецификации см. контроллеры в `backend/src/main/java`.

Общие правила:

- Все эндпоинты находятся под базовым путём, настроенным в `application.yml`.
- Авторизация: JWT / сессии (проверь `src/main/java` на реализацию).

Примеры (примерные пути — уточнить по коду):

- `POST /api/auth/login` — вход пользователя
- `POST /api/auth/register` — регистрация
- `GET /api/subjects` — список предметов
- `POST /api/materials` — загрузка образовательных материалов

Форматы запросов/ответов:

- Используется JSON. См. DTO в коде для структуры поля.
