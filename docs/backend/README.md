# Backend — Сервис (Spring Boot)

Бэкенд находится в `backend/` и реализован на Spring Boot.

- Основные технологии: Java 17+, Spring Boot, Maven
- Сборка: `mvnw package` (используется wrapper)
- Точка входа: класс в `src/main/java/...` согласно `pom.xml`

См. также:

- `setup.md` — требования и среда
- `development.md` — запуск и локальная отладка
- `api.md` — описание REST API
- `deployment.md` — сборка и развёртывание (Docker)
