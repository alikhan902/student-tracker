# Desktop — Разработка и отладка

Запуск из IDE:

- Открой `desktop/` как Maven-проект в IntelliJ IDEA или другой IDE.
- Точка входа — класс с `public static void main` (поиск по проекту).

Запуск через Maven:

   ./mvnw javafx:run

Где искать ресурсы UI:

- FXML и CSS находятся в `desktop/src/main/resources/` (`fxml/`, `css/` и `student_tracker/`).
