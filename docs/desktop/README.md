# Desktop — Десктопное приложение

Десктопное приложение находится в каталоге `desktop/`.

- Технологии: Java, JavaFX, Maven
- Сборка: `mvnw package`
- Запуск: `java -jar target/desktop-*.jar` или через IDE

См. также:

- `setup.md` — настройка окружения
- `development.md` — разработка и отладка
- `testing.md` — unit/UI тесты
