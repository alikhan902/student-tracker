# Оценка трудоёмкости (COCOMO II — упрощённая)

## Исходные данные

| Параметр | Значение |
|----------|----------|
| Размер кодовой базы (оценка SLOC) | ~15 000 (backend ~8k, frontend ~5k, desktop ~2k) |
| Режим | Semi-Detached (средняя сложность, знакомый стек) |
| Команда | 1 разработчик (учебный проект) |

## Базовая формула (COCOMO I, organic как ориентир)

`Effort = a × (KLOC)^b`  
При organic: a≈2.4, b≈1.05 → для 15 KLOC: **~41 чел.-дня** (сырая оценка).

## COCOMO II (Application Composition — грубая оценка)

| Web pages | 6 | средний |
