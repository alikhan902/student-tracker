# Руководство пользователя — Student Tracker

## 1. Введение

**Student Tracker** — система учёта учебных материалов для студенческой группы.  
Доступ: **веб-браузер** (`http://localhost` через nginx) или **десктопное приложение** (JavaFX).

---

## 2. Начало работы

### 2.1. Регистрация

1. Откройте `/signup`
2. Укажите имя, логин, email, пароль
3. Выберите тип: **Студент** или **Староста**
4. Нажмите **«Создать аккаунт»**

**Альтернатива:** кнопка **«Войти через Google»** (веб).

### 2.2. Вход

1. Страница `/login` — логин и пароль
2. После входа откроется **Профиль**

### 2.3. Восстановление пароля

1. **«Забыли пароль?»** → email
2. Перейдите по ссылке из письма
3. Задайте новый пароль

---

## 3. Профиль

- Просмотр и редактирование имени
- Смена пароля (для локальных аккаунтов)
- Удаление аккаунта

---

## 4. Учебная группа (староста)

1. Раздел **«Группа»** в меню
2. **Создать группу** — введите название
3. **Добавить студента** — логин уже зарегистрированного пользователя типа «Студент»
4. **Удалить студента** — из списка участников
5. **Удалить группу** — при необходимости

> Студент без группы видит приглашение дождаться добавления старостой.

---

## 5. Учебные предметы

### Староста

1. **«Предметы»** → **«Создать предмет»**
2. Название, описание, фото (опционально)
3. Открыть предмет → управление материалами

### Студент

- Просмотр списка предметов группы
- Открытие карточки предмета

---

## 6. Образовательные материалы

### Староста

- **Добавить материал:** название, описание, файл (PDF и др.)
- Редактирование и удаление

### Все участники

- **Скачать** прикреплённый файл
- **Отметить выполнение** — переключатель «Изучено» / «Не изучено»

---

## 7. Десктопное приложение

1. Соберите и запустите JAR (см. [заметки по реализации](../05-implementation/implementation-notes.md))
2. Экраны: вход → профиль / предметы / группа
3. Требуется запущенный backend на `localhost:8080`

---

## 8. Частые проблемы

| Проблема | Решение |
|----------|---------|
| 401 на API | Выйдите и войдите снова (истёк JWT) |
| Не видно предметов | Убедитесь, что вы в группе |
| OAuth не работает | Проверьте `GOOGLE_CLIENT_*` и `APP_FRONTEND_URL` |
| Файл не открывается | Проверьте доступ к `/uploads/...` через nginx |
