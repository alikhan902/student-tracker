# Frontend — Настройка окружения

Требования:

- Node.js 18+ (рекомендуется 18.x/20.x)
- npm или pnpm

Установка зависимостей:

1. Перейти в папку проекта:

   cd frontend

2. Установить зависимости:

   npm install

Переменные окружения:

- В корне `frontend/` может находиться `.env` файл. Пример значений описан в корневом `.env.example`.
