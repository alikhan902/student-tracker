# Frontend — Веб-приложение

Этот раздел описывает фронтенд-приложение, расположенное в каталоге `frontend/`.

- Технологии: React, Vite, Tailwind CSS, Vitest
- Основные скрипты: `npm run dev`, `npm run build`, `npm run test`

См. также:

- `setup.md` — настройка окружения
- `development.md` — запуск и отладка
- `deployment.md` — сборка и деплой
