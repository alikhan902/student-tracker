# Frontend — Сборка и деплой

Сборка production-версии:

1. В каталоге `frontend` выполнить:

   npm run build

2. Результат будет в `dist/`.

Деплой:

- В проекте настроен `nginx/` для проксирования статических файлов.
- Можно развернуть контейнер с `Dockerfile` в `frontend/` или копировать содержимое `dist/` в выбранный хостинг.
