# BUC-диаграмма (бизнес-прецеденты)

## Диаграмма

**PlantUML:** [`images/00/business_precedents.puml`](../images/00/business_precedents.puml)

## Границы ответственности

| Внутри системы | Вне системы |
|----------------|-------------|
| Хранение пользователей, групп, предметов | Регистрация в Google/GitHub (OAuth-провайдер) |
| Выдача JWT, проверка прав | Доставка email (SMTP-провайдер) |
| Загрузка и раздача файлов материалов | Браузер ОС пользователя |
| Учёт статусов выполнения | Локальная ФС при скачивании файла на desktop |
