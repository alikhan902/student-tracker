# Матрица стейкхолдеров

| Стейкхолдер | Интерес | Влияние | Вовлечённость | Стратегия |
|-------------|---------|---------|---------------|-----------|
| Староста (HEADMAN) | Удобное управление группой и материалами | Высокое | Ежедневно | Приоритет UX для CRUD предметов/материалов |
| Студент (STANDARD) | Быстрый доступ к материалам и отметкам | Среднее | Ежедневно | Простой список предметов и toggle выполнения |
| Преподаватель (оценщик) | Соответствие МУ, документация, тесты | Высокое | По этапам сдачи | Полный комплект docs/, Swagger, Docker |
| Разработчик (автор) | Сроки, качество кода | Высокое | Постоянно | PCMEF, приоритет core-функций |
| Администратор инфраструктуры | Стабильный деплой | Среднее | При релизе | docker-compose, nginx, .env |

## Матрица влияния / интереса

```
        Высокий интерес
              │
   Староста   │   Преподаватель
   Студент    │
──────────────┼──────────────  Высокое влияние
              │
   Аудитор    │   DevOps
              │
        Низкий интерес
```
