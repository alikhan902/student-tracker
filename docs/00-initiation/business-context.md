# Диаграмма бизнес-контекста (IDEF0 A-0)

## Контекстная диаграмма

**PlantUML:** [`images/00/business_IDEF0.puml`](../images/00/business_IDEF0.puml)

## Описание

**Название функции:** Управление учебными материалами студенческой группы  
**Уровень декомпозиции:** A-0 (контекстный)

### Входы

| Вход | Источник |
|------|----------|
| Запросы на вступление в группу | Студент (STANDARD) |
| Название группы, состав | Староста (HEADMAN) |
| Файлы и описания материалов | Староста |

### Выходы

| Выход | Получатель |
|-------|-----------|
| Структурированные предметы и материалы | Все участники группы |
| Статусы COMPLETED / NOT_COMPLETED | Студент, староста |
| Письма сброса пароля | Пользователи |

### Управляющие воздействия

- Тип студента: `HEADMAN`, `STANDARD`
- Роли системы: `ADMIN`, `USER`, `AUDITOR`
- Политики: JWT, BCrypt, HTTPS

### Механизмы исполнения

- Веб-браузер + React (Vite)
- JavaFX Desktop
- Spring Boot REST
- PostgreSQL 16

## Декомпозиция A0 → A1..A5

| Код | Подфункция |
|-----|-----------|
| A1 | Управление учётными записями (регистрация, вход, OAuth2, профиль) |
| A2 | Управление учебной группой и составом |
| A3 | Управление учебными предметами (training subjects) |
| A4 | Управление образовательными материалами (файлы) |
| A5 | Учёт выполнения материалов (material complete) |
